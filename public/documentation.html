My Flix

What this API does:

This app will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Movie Endpoints and Methods

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie- description, genre, director
Return data about a genre by name/title /movies/genre/:genreName GET None A JSON object holding data about a genre
Return data about a director /movies/directors/:directorName GET None A JSON object holding data about a director: name, biography, birth year, and death year

User Endpoints and Methods

Request URL HTTP Method Request Body Data Format Response Body Data Format
New user registration /users POST A JSON object holding data about the new user like: { username: "petersmith" } A JSON object holding data about the new user that was added, including an ID: { id: "" username: "petersmith" }
Update a user's username /users/:id PUT None A JSON object holding new data (the new username) about the updated user
Add a movie to user's list of favortie movies /users/:id/:movieTitle PUT None A text message indicating the new data (new movie) has been added to the user's list
Delete a movie from the user's list of favorite movies /users/:id/:movieTitle DELETE None A text message indicating the data (movie) has been deleted from the user's list
Delete an existing user /users/:id DELETE None A text message indicating the user has been deleted
